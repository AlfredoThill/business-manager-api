### Installation

1. Install NPM packages
    ```sh
    npm install
    ```
2. Create `.env` file in main directory, from `.env.example`
3. Test and build the project
    ```sh
    npm run build
    ```
4. Sync database tables, if necessary
    ```sh
    npm run sync-db
    ```
5. Run the server
    ```sh
    npm run start
    ```
6. Access swagger docs from `localhost:8081/docs/v1`
